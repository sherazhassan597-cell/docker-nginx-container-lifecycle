# docker-nginx-container-lifecycle
## Project Overview 
This project demonstrates Docker container lifecycle management including:
- Pulling image from Docker Hub
- Running container in detached mode
- Listing running and stopped containers
- Remove containers \
  ## Commands Used
  ```bash 
  # pull nginx
  dcker pull nginx
  # Run Continers in Detached mode
  docker run -d nginx
  # Check Containers
  docker ps -a
  # Remove Container
  doocker rm <container_id>
  ```
